# Gummy-Todo

Gummy-Todo is a simple TUI writen in bash (Gum)

## Installation

In order to use it, you need:
- [Go installation](https://go.dev/doc/install)
- [Gum installation](https://github.com/charmbracelet/gum)



## Usage

```bash
./main.sh
```
